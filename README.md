# Bluebird Trail Project

This repository contains the data, planning documents, and project logs for the Bluebird Trail monitoring initiative, alongside various technical research reports and personal journal entries.

## Repository Contents

### Bluebird Monitoring Data

The core of this repository consists of nest box monitoring data and planning documents for the Flint Preserve. The data is organized into several CSV files representing different monitoring dates, alongside a comprehensive planning summary.

| File Name | Description |
| :--- | :--- |
| `bluebird_monitoring_June23_2025.csv` | Nest box monitoring data recorded on June 23, 2025. |
| `bluebird_monitoring_July21_2025.csv` | Nest box monitoring data recorded on July 21, 2025. |
| `bluebird_monitoring_Oct16_2025.csv` | Nest box monitoring data recorded on October 16, 2025. |
| `bluebird_nest_observations.csv` | General nest observations and historical data. |
| `Flint_Preserve_Nest_Box_Planning_Summary.md` | A quantitative analysis of historical nest box survey data (2021–2022) and a proposed architecture for a mobile-first field recording workflow. |

### Project Logs & User Preferences

This section documents the ongoing projects, personal preferences, and daily activities of the user, Bill Kaiser. It includes a detailed log of tasks accomplished using Manus, personal communication styles, and journal entries detailing property maintenance and gardening.

| File Name | Description |
| :--- | :--- |
| `manus_project_log.md` | A comprehensive log of projects built, configured, and accomplished using Manus, including Google Sheets automation, contractor research, and local LLM setup. |
| `soul.md` | User preferences, communication style, and working philosophy for interacting with AI agents. |
| `Bill_Kaiser_Journal.md` | Personal journal entries detailing daily tasks, gardening (planting peonies), and maintaining the bluebird trail infrastructure. |

### Technical Research & Reports

The repository also houses several technical reports and practical guides exploring the capabilities of local LLMs, AI agent platforms, and AI-powered interview preparation tools.

| File Name | Description |
